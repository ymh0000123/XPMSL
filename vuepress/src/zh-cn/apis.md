# XPMSL的api
## 1. 获取版本
```
https://xpmsl.pages.dev/releases.json
```  
| 参数 | 描述 |
| ---- | ---- |
| releases | XPMSL releases版本 |
| build | 版本号|  
| up_release | 更新程序releases版本 |  
| up_build | 更新程序版本号 |
